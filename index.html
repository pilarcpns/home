Pertanyaan yang Sering Ditanya

Delapan pertanyaan umum sebelum Anda membeli.

Format file apa yang dikirim?

PDF. Bisa dibuka di HP, laptop, atau dicetak A4. Tidak perlu install aplikasi tambahan. Disarankan menggunakan Adobe Acrobat Reader atau browser Chrome untuk pengalaman membaca terbaik.

Apakah modul bisa dicetak?

Ya, format A4 hitam putih. Hemat tinta, hasil tetap jelas. Cocok untuk latihan dengan timer seperti simulasi CAT sungguhan.

Pembahasannya seperti apa?

Setiap soal ada pembahasan detail, bukan hanya kunci jawaban. Untuk TWK dan TIU ada penjelasan logika. Untuk TKP ada scoring 5-1 per pernyataan.

Berapa lama akses setelah beli?

Selamanya. Akses permanen, tidak ada batas waktu. File PDF bisa disimpan di perangkat Anda dan digunakan kapan saja tanpa expire.

Bagaimana cara ordernya?

Lewat lynk.id untuk pembayaran otomatis (QRIS, transfer bank, e-wallet). File dikirim ke email dalam hitungan menit setelah pembayaran. Untuk paket custom, gunakan email pilarcpns@gmail.com untuk konsultasi.

Apakah bisa refund?

Karena ini produk digital dan file langsung dikirim setelah pembayaran, refund tidak tersedia. Tersedia sample gratis satu modul penuh untuk cek kualitas sebelum membeli.

Update gratis berlaku untuk apa?

Jika ada revisi soal, perbaikan pembahasan, atau penambahan modul baru, semua pembeli lama akan mendapat update gratis tanpa biaya tambahan.

Apakah ada jaminan lulus?

Tidak ada yang bisa menjamin kelulusan. PilarCPNS adalah alat bantu latihan. Hasil tetap bergantung pada usaha dan persiapan Anda.

Siap Lolos Tes PNS?

Mulai latihan dari sekarang. Lebih dari 1.199 soal dengan pembahasan lengkap sudah menunggu.
